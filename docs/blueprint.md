# **App Name**: TripVision

## Core Features:

- API Key Input: Securely capture the user's Gemini API key.
- Trip Preferences Form: Collect the user's travel preferences, including start date, destination, trip duration, budget, and interests.
- AI-Powered Itinerary Generation: Use the Google Gemini API to generate a personalized trip itinerary based on user preferences, and includes dynamic insights on travel availability and pricing considerations. The AI acts as a tool to enhance the itinerary.
- Itinerary Display - List View: Display the generated itinerary in a clear, readable, day-by-day list format with a welcoming placeholder, a title showing the destination and an AI insight section that features forecast analysis.
- Itinerary Display - Calendar View: Render the itinerary in a monthly calendar format using react-big-calendar, highlighting trip days and displaying activities as events, that each expand with a description and cost when clicked. Makes use of the user's start date.
- View Toggle: Prominent buttons to switch between 'List View' and 'Calendar View'. The active view's button should be styled differently to indicate it's selected.

## Style Guidelines:

- Primary color: Sky blue (#7EC8E3), evoking calmness and clarity for travel planning.
- Background color: Light off-white (#F9F9F9), providing a soft, unobtrusive backdrop.
- Accent color: Warm coral (#FF7F50), used for interactive elements and highlights, promoting a friendly feel.
- Body and headline font: 'Inter', a grotesque-style sans-serif font offering a modern, objective feel for excellent readability.
- Mobile-first, responsive design with a two-column layout on larger screens for optimal viewing on all devices.
- Use clear, simple icons to represent different activities and interests.
- Implement smooth transitions and subtle animations to enhance interactivity.
